#### AI能做什么
`AI`的新闻经常登上热搜，架势像极了90年底的`互联网`潮。我们的生活方式，因为技术革新发生了显著改变，很多企业、组织通过拥抱互联网得以发展壮大。技术革新的本质是去`差异化`，甚至有些普度众生的感觉。新的技术趋势到来的时候，新一代企业家、公司高管都在思考：人工智能，到底能为她的企业创造什么价值。我们听过太多`AI的传奇故事`，但Alpha-GO在AI知识普及上并没有起到好的表率。

作为`机器学习`的分支，人工智能很大程度上解决的是`输入到输出`的优化，所以Google会把每年的开发者大会称为`I/O大会`。我们用`X`定义输入，`Y`定义输出，人工智能是解决知道X，如何得到Y的问题。在这个系统框架下，人工智能可以改变很多行业。特别是当这个行业的具体工作、场景，可以通过人工的简单判断就得出结论的。但这样的改变，一定是逐步迭代、优化，而不是一个"见证奇迹的时刻"。下面是一些已经实现的例子：

| 输入（X）                    | 输出（Y）            | 应用场景     |
| ---------------------------- | -------------------- | ------------ |
| 基于血蛋白提取的光谱数据     | 患者是否有某种疾病   | 疾病早期诊断 |
| 特朗普总统的演讲录音         | 总统先生的演讲稿脚本 | 语音转文字   |
| 总统先生的英文演讲稿         | 总统先生的日文演讲稿 | 翻译         |
| 一张宠物照片                 | 宠物是猫还是狗       | 图像识别     |
| 客户提出的一个产品相关的问题 | 给出该问题的答案     | 自然语言处理 |

#### AI不能做什么
从X到Y的转化过程可以看出，这个系统的建设需要很多X和Y，以及X->Y的关系。专业术语将这个转化过程、或系统称为`专家学习系统`。如今，相关人工智能技术仍在高速的迭代，大家谈论更多的是`深度学习`，`深度神经网络`等技术在上述场景中的应用，这些算法的名字也让很容易让大家联想到人的大脑。但事实上，人类智能能胜任的工作，是远远超过哪怕是最新的专家系统；说AI替代人类，实在为时过早。

即便如此，为了让这个专家系统投入产出，我们需要喂给它很多数据。是的，很多、很多的数据，并且X和Y的数据都要。笔者从业经历看，获得这些数据的成本并不便宜，或者你需要一群人配合你做大量的数据收集、处理工作，`AI训练师`的岗位顺应而生，后面我们有专门的章节讲这个。

判断你的工作、所在的行业或企业，AI是否适用的标准是什么？你会因为AI技术的迭代而失业吗？我们在这介绍一个简单的方法，称为`杰克船长法`。迪斯尼系列电影《加勒比海盗》中的杰克船长，经常在危难之时要判断下一步的重大方向，他的标准动作是：

1. 第一步：爬到帆船桅杆的高处，取出望远镜观察远方的环境。
2. 第二步：放下望远镜，环顾海盗船上的人力、武器、供给。
3. 第三步：作出决定，按照望远镜看到的目标前进或调整方向。

作为企业家、公司高管、团队负责人，你是企业、组织和团队的`领航员`，`Nagvigator`。在判断是否采用AI技术的时候，可以采取如下步骤：

1. 第一步：判断你想解决的业务问题、流程或场景，能否通过简单判断就可以处理。如果是，这项工作或业务就有被AI优化的可能。
2. 第二步：在第一步选定的场景里，是否已经有足够的X，Y，X->Y的数据，以及处理X，Y数据的技术人才。如果当前没有数据和人才，获取的成本高吗？如果当前就有，或容易获得，那你采用AI技术的可能性，又增加了很多。
3. 第三步：Go或No-Go。对AI适用的业务，应制定对应的业务目标，评估投产比；对AI不适用的业务，我们可以继续寻找新的方向。

找到适用AI的场景不难，大部分企业往往在第二步出错，主要表现在：
1. 没有充分预估数据质量差，带来项目失败的影响。
2. 缺少合适的数据分析、算法人才，无法将通用的AI算法调优，适应自身企业的需求。
3. 对AI所需的持续运营投入没有心理预设，项目上的快、下的更快。
